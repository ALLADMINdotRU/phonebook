Как работать с картой
Как разместить пользователей на карте:
  1. Нажмите кнопку "Разместить" рядом с нужным пользователем
  2. Кликните на карте в том месте, где должен быть пользователь
  3. Маркер появится автоматически - координаты сохранятся
Управление маркерами:
  • Кликните на маркер пользователя чтобы увидеть информацию
  • Нажмите кнопку "Убрать с карты" чтобы удалить размещение
  • Используйте поиск для быстрого нахождения пользователей
Важно:
  • Сначала загрузите план здания через редактирование сервера
  • Координаты автоматически сохраняются при размещении
  • Для изменения позиции - уберите и разместите заново
